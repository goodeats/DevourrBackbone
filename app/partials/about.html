About the Developer

This is an opensource front end and back end project created by Pat Needham

This was previously an Angular.js project which I have since reverted back to Backbone.js and Handlebars. While I liked some of the structure that Angular provided (my personal website uses Angular), I found some of its rules a bit too strict for a framework that will eventually not be supported. I will look to explore React.js for my next project as I hear good things.
